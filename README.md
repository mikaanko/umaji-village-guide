# Umaji Village Travel Guide

Draft English tourism guide site for international visitors to Umaji Village.

## Structure

- `app/page.tsx`: Top page
- `app/spots/page.tsx`: Spot list page
- `data/spots.ts`: Replaceable draft spot data
- `components/SpotCard.tsx`: Spot card with category, notes, map button, and photo placeholder

## Local Development

```bash
npm install
npm run dev
```

All current spot information is temporary draft data. Replace names, photos,
hours, fees, notes, and map links after confirmation.
