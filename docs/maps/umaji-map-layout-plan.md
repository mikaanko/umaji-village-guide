### 座標から確定できる並び

- 北から南: 馬路森林鉄道・インクライン → 馬路温泉 → やまなみ食堂 → 馬路村役場 → うまじのパン屋 → 柚子の森加工場 → 金林寺 → まかいちょって家 → 五味隧道。

### SVG座標への変換式と検証値

中心部本図の登録9点には、以下の同一線形変換を使用する。SVGでは右が `x` 正方向、下が `y` 正方向なので、緯度だけは基準緯度から引く。

```text
x = 275 + (経度 - 134.0466897) / (134.0494737 - 134.0466897) × 305
y = 205 + (33.5566774 - 緯度) / (33.5566774 - 33.5492511) × 715
```

追加点にも既存点と同じ縮尺を適用する。森林鉄道・インクライン共有点は従来の北端だった温泉より北にあるため、同式により温泉の `y=205.0` より小さい `y=176.5` となる。既存7点のSVG座標は変更しない。

- 経度が大きいほど `x` が大きくなり、右へ移動する。
- 緯度が大きいほど `y` が小さくなり、上へ移動する。
- 朝日出山の大杉は中心部と同じ縮尺にすると本図が極端に圧縮されるため、別枠小地図の `(635, 1262)` に置く。これは方向・距離感用で、上式の投影対象外。

#### 修正前に位置関係が判定しづらくなった原因

初版は座標値を確認していたものの、SVGピン座標を共通の変換式から生成せず、見た目に合わせた概算値を直接記入していた。さらに河川も概略曲線だったため、ピン・ラベル・川岸のどれを位置基準とするかが曖昧だった。修正版では、ピン中心だけを座標投影の基準とし、全ピンに `data-lat`、`data-lng`、`data-svg-x`、`data-svg-y` を付けて検証可能にした。

## 2. Googleマイマップ拡大表示による川岸・道路の再検証

地点間の経度比較だけでは川岸を判定できないため、Googleマイマップを中心部まで拡大し、各地点の緯度で安田川中心線がピンの東西どちらに見えるかを確認した。

### 温泉・役場区間で誤って見えた原因

初版の安田川は、地図上部から下部までほぼ同じ経度を通る緩い縦線として描いていた。しかし実際の安田川は、温泉付近では東へ張り出し、その直後に役場の西側まで大きく曲がる。この強いS字を省略したため、正しい緯度経度のピンを置いても川岸だけがGoogleマップと異なって見えた。

修正版ではスポット座標を維持し、河川中心線を次のように変更した。

1. 温泉の緯度では、河川中心線を温泉ピンより右（東）へ通す。
2. 温泉から役場へ南下する間に、河川を大きく左（西）へ曲げる。
3. 役場の緯度では、河川中心線を役場ピンより左（西）へ通す。
4. 役場より南では再び東寄りへ戻し、パン屋・加工場・金林寺の東側を通す。
5. まかいちょって家・五味隧道周辺では再度東へ張り出す曲線にする。
6. 主要道路12も河川の東側を基本に、同じS字へ追従させる。

### 表示上の位置調整

- まかいちょって家は、座標投影位置 `(580.0, 872.0)` が概略化した太い河川線と重なって見えるため、表示用ピンのみ東へ55px移動し `(635.0, 872.0)` とした。
- 登録緯度経度と本来の投影座標はSVGの `data-lat`、`data-lng`、`data-projected-svg-x`、`data-projected-svg-y` に保持する。
- この調整は白地図ラフの判読性のためで、他スポットの座標投影や川・道路形状は変更しない。

## 4. 地図の構成案

1. **縦長の中心部本図**
   中心部7登録点を北から南へ配置する。安田川を中央軸にし、川沿いの主要道路を一本に整理する。道路や川の細かな蛇行は案内図用に簡略化する。
2. **中心部は実距離比を土台にラベル間隔だけ調整**
   ピン位置は座標比を保ち、ラベルは引き出し線で左右に逃がす。特にパン屋・加工場、まかいちょって家・五味隧道を重ねない。
3. **朝日出山の大杉は別枠小地図**
   「中心部から南東、車で移動する郊外スポット」として方向と距離感を示す。中心部本図と同縮尺にはしない。
4. **共有登録点は名称を分けて表示**
   やまなみ食堂は確定ピンへ変更する。森林鉄道とインクラインはマイマップ上の共有ピンを維持し、1本の引き出し線の先で2名称を併記する。
5. **将来のイラスト追加余白**
   西右・東左に固定せず、川の両側へスポットアイコン約56–72pxを置ける余白を残す。山林は外周、建物イラストは中心部ラベル周辺へ追加する。

## 5. 白地図ラフの読み方

- **実線の丸ピン**: Google マイマップ登録座標を基準にした位置。
- **破線枠**: 郊外小地図など、中心部本図と縮尺が異なる領域。主要スポットに座標未取得の仮ピンは残していない。
- **青線**: 安田川の概略軸。正確な河道形状の再現ではない。
- **茶灰線**: 川沿いの主要道路の概略。ユーザー指定は「国道12号」だが、マイマップ画面では「12」の道路番号のみ確認できるため、道路種別と正式名称は要確認。
- **緑の外周面**: 山・森林の背景領域。完成時に淡い植生イラストへ置換可能。

## 6. 要確認箇所

1. 森林鉄道とインクラインを将来別々のピンにする場合は、それぞれの個別座標が必要。現マイマップでは1つの共有登録点。
2. Aコープ、森林組合、馬路中学校、JA本所の個別座標と表示優先度。
3. 地図に描く道路の正式名称。「国道12号」か「高知県道12号」等かを一次資料で確認する。
4. 安田川の正確な左右関係。座標点だけでは各施設が河川の東西どちら岸かを確定できない箇所がある。
5. 「馬路温泉」「うまじ温泉」、「柚子の森加工場」「ゆずの森加工場」の最終表記。
6. 添付スクリーンショット原寸で、中心部の川・道路・橋・補助施設の位置を再確認する。
